# Code of Conduct

이 프로젝트는 [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
을 따릅니다.

요약:
- 존중하는 의사소통. 기술적 비판은 환영, 개인 공격은 금지.
- 다양한 배경·경험·관점의 기여자를 환영.
- 건설적 비판을 수용하고, 실수에서 배우고, 사과할 수 있어야 함.
- 커뮤니티 전체의 이익을 개인 이익보다 우선.

### 금지 행위

- 성적 언어·이미지, 원치 않는 접근
- 트롤링, 모욕, 정치적·개인적 공격
- 공개 또는 사적 괴롭힘
- 동의 없이 타인의 개인정보 (이메일, 주소 등) 공개
- 전문적 환경에서 부적절한 기타 행위

### 적용 범위

프로젝트 공간 (이슈, PR, 리뷰, 커밋 메시지) 과 공개 장소에서의 프로젝트
대표 활동 모두.

취해집니다. 신고자의 신원은 기밀로 유지됩니다.
